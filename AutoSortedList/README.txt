auto_sorted_list.cpp
This file creates a constructor and defines a series of operations used in the main file.

auto_sorted_list.h
This file creats a class for our list with variables each item has as well as providing a way to call the operations created in the a_s_l.cpp file.

node.h
This file provides a way to keep track of our list through creating nodes for the variables in the list and keeps them in a sequential order.

node.cpp
This file offers us a way to start our list and provides a way for us to keep track of the order of variables in the list.

main.cpp
Takes in a group of variables and places them in a list in sequential order. Next prints the length, adds the variable 10 as a list item, and prints off the length afterwards.
